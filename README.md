# Simon Game

## Overview
The Simon Game is a classic memory game where players must repeat a sequence of light and sound signals. The game tests players' memory and attention, providing an engaging way to enhance cognitive skills.

## Game Rules
1. The game starts by displaying a sequence of colors and sounds. 
2. Players must memorize the sequence and reproduce it by clicking the corresponding colored buttons.
3. With each successful repetition, the sequence gets longer.
4. If a player fails to reproduce the sequence correctly, the game is over.

## How to Play
1. Start the game by clicking the "Start" button.
2. Watch the sequence of colors displayed on the screen.
3. After the sequence is shown, click the colored buttons in the same order.
4. Repeat the sequence as the game progresses.
